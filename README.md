# game_protocolo_fantasma_25_12_03
PROTOCOLO FANTASMA: Desafío de enigmas Sci-Fi/Neón para salvar la Simulación. Siete constantes (matemáticas, física y lógica) deben ser resueltas por el Agente Falk.
Proyecto: PROTOCOLO FANTASMA (The Ghost Protocol)

Un juego de escape room digital de ciencia ficción basado en web (HTML, CSS, JavaScript) con estética neón. El Agente Falk Artis Bay debe descifrar una secuencia de siete constantes (basadas en Termodinámica, Mecánica Cuántica, Electrónica y Matemáticas) para activar el "Reinicio Maestro" y detener el colapso del Núcleo de la Simulación.

Tecnologías: HTML5, CSS3 (Efectos Glitch/Neón), JavaScript (Lógica de enigmas y persistencia con localStorage).

Objetivo: Restaurar el equilibrio entre las inteligencias absolutas, Raoul (Permanencia) y Erik (Evolución).
